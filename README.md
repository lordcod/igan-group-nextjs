


# ИГАН Групп — Конфигуратор алюминиевых фасадов

Проект на [Next.js](https://nextjs.org/) с клиентским конфигуратором фасадов: выбор профиля, анодирования, стекла и фурнитуры с расчётом цены в реальном времени.

---

## 📦 Функциональность

- Конфигуратор алюминиевых фасадов с визуализацией размеров
- Выбор профиля, цвета анодирования, стеклянного заполнения и фурнитуры
- Динамический расчёт стоимости (включая доставку и сборку)
- Удобный интерфейс и мобильная адаптивность
- SSR, оптимизация загрузки и готовность к продакшену

---

## 🚀 Технологии

- **Next.js 14+ (App Router)**  
- **React 18**  
- **Tailwind CSS**  
- **Vercel Hosting** (https://igan-group-nextjs.vercel.app)

---

## 📦 Установка

```bash
# Установка зависимостей
npm install

# Запуск в dev-режиме
npm run dev

# Билд и предпросмотр
npm run build
npm run start
````
